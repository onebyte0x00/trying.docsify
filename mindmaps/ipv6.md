# IPv6 Fundamentals
```mermaid
mindmap
  root((IPv6))
    Address Format
      128-bit, 8 groups of hex
      Example: 2001:0db8:85a3::8a2e:0370:7334
    Address Types
      Unicast: Global, Link-Local
      Multicast: One-to-many
      Anycast: Nearest receiver
    Key Features
      No NAT needed
      Built-in security (IPsec)
      Autoconfiguration (SLAAC)
    Transition Technologies
      Dual Stack: Runs IPv4 + IPv6
      Tunneling: 6to4, GRE
