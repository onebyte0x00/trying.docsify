# TCP/IP Model (4 Layers)
```mermaid
mindmap
  root((TCP/IP Model))
    Application Layer
      Combines OSI: App, Presentation, Session
      Examples: HTTP, SSH, DHCP
    Transport Layer
      TCP: Reliable, ordered
      UDP: Unreliable, fast
    Internet Layer
      IP: IPv4, IPv6
      Routing: OSPF, EIGRP
    Network Access
      Combines OSI: Data Link + Physical
      Ethernet, Wi-Fi, PPP
